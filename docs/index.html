un-PC

un-PC is a quick-to-run, intuitive way to visualize patterns of genetic differentiation across a landscape.

Created by Geoffrey House and Matthew Hahn (Indiana University)

un-PC is published as part of: House and Hahn Evaluating methods to visualize patterns of genetic differentiation on a landscape. Molecular Ecology Resources (2018) doi:10.1111/1755-0998.12747

Download and install un-PC from GitHub

For an overview of how to run un-PC on existing PCA results, see the guide here: un-PC tutorial.
